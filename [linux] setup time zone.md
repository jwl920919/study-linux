# TimeZone 설정

### TimeZone 변경
```
rm /etc/localtime
ln -s /usr/share/zoneinfo/Asia/Seoul /etc/localtime
```

### TimeZone 및 시간 확인
```
date

$ 2016. 08. 09. (화) 10:26:44 KST
```
